class smaug::ReferenceBackend¶

ReferenceBackend provides reference implementations of all operators supported by SMAUG. Some of these operators are written to be Aladdin-compatible for simulation purposes; others are intended to be executed in software. As the name suggests, these implementations are not optimized in any way.
